# Weather API Console Application

> Simple console application that makes an API call request to WeatherAPI.com

![](weather.png)

## Installation

Windows:

```sh
* Download zip file

* Open CMD or powershell and CD into the WeatherAPI/bin/Release/net6.0

* Run the WeatherAPI.exe
```
